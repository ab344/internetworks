How the Internet Works

"The Internet, a loosely-organized international collaboration of autonomous, interconnected networks, supports host-to-host communication through voluntary adherence to open protocols and procedures defined by Internet Standards."

This website has been designed as a project for the Web Systems Development course, IS-601 at NJIT. The purpose of the website is to provide information about the internet.

Topics discussed in this project include:

Domain Names and ICANN IP Addresses, Packets, and Routing DNS HTTP and HTTPS protocols W3C, HTML, and CSS

To learn more about this project, please visit the About page.
